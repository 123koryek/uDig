


Use a bookmark as the AOI
~~~~~~~~~~~~~~~~~~~~~~~~~

We can use an existing bookmark to define the current AOI.


#. Open the **AOI** view
#. Select the **Bookmark** option form the **AOI** combo box
#. Select the desired bookmark from the **Bookmarks** combo box
