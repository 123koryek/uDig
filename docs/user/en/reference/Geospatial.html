Geospatial

Geospatial concepts and terminology concern geographic (often terrestrial) information.

As a Geographic Information System uDig makes use of many geospatial terms. The informal definitions contained in our online help are intended as an introduction to these ideas, please consult the cited reference material if you are interested in learning more.
