Printing the current Map

To print the current map you must first create a Page that allows you to lay it out and print it from there.

  1. Open the map you wish to print
  2. Press the print button on the toolbar to create a page (or select File > Print )
  3. It will ask you what template you wish to create the page from.
  4. Page Editor will be opened showing the arrangement of the Page
  5. You can now press the pdf button to generate a PDF
