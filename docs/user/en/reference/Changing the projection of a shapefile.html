Changing the projection of a shapefile

In general uDig lets you work with shapefiles of different projections all on the same screen together. If you need to save a shapefile out in a specific projection for another application please use the following procedure.

  1. Select the contents of your shapefile in the Catalog view; or by selecting a Layer if you have the shapefile on screen.
  2. Open the Export wizard and choose Resource to Shapefile and press Next>
  3. Chose a destination directory for your new shapefile:
    C:\Documents and Settings\Jody\Desktop\data
  4. Ensure your shapefile is checked off on in the list
  5. Click on the current projection, example GCS_EGS1984, to open up a CRS chooser
  6. Type in the name of the desired projection, example UTM zone 12N, and select the desired EPSG code from the list. You can confirm the details by looking at the Custom CRS tab
  7. Press Finish, your newly created shapefile is available in the Catalog view
