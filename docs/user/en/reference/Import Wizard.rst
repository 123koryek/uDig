


Import Wizard
~~~~~~~~~~~~~

This wizard allows you to import additional information into the uDig
application.



You can choose between the following options:


+ `CSV Import page`_
+ `Data page`_
+ `Import imagery folder as mosaic page`_
+ `KML to Shapefile import page`_



+ `Importing data`_



+ `Catalog view`_
