Layer Menu

The Layer Menu is used to control the content displayed in the Map Editor. Many of the actions available in the Layer menu apply to the selected Layer as shown in the Layer view.

Add

Opens the Add Data wizard to add additional layers to the current map.

Create

Opens the Create Data wizard to create custom layers for the current map.
