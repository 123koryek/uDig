


Show the AOI on the map
~~~~~~~~~~~~~~~~~~~~~~~

We can add a map decoration to visually show the extent of the current
AOI.


#. Add a new layer to the existing map
#. Select **Map Decoration** and click **Next**
#. Check **AOI** and click **Finish**
