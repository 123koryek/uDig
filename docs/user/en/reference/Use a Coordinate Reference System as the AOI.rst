


Use a Coordinate Reference System as the AOI
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

We can use the extent of the map Coordinate Reference System to define
the current AOI.


#. Open the **AOI** view
#. Select the **Reference System** option form the **AOI** combo box
