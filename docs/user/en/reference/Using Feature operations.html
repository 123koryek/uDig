Using Feature operations

Feature operations are available from a number of locations:

  • Listed in the edit menu when a features are selected.
  • from the context menu when features are selected
  • from the menu bar Edit > All Operations to open the Operations dialog

There are several common ways to select features to work on.

  • Using the Catalog view to open up a service and select the features being published
  • Using the Map editor and Selection Tools to indicate which features to work on
  • Using the Layers view to select a layer (the layer selection will be used if provided)
