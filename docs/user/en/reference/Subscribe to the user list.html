Subscribe to the user list

An email list is available for questions and comments on the uDig application.

  1. Navigate to the following page:
    http://lists.refractions.net/mailman/listinfo/udig-users
  2. Enter your email address, name and password
  3. Press the Subscribe button
  4. The system will email you a confirmation message, reply to this message to subscribe to the list
