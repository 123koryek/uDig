Enabling a Tileset from a WMS Server

Enabling tilesets on a wms

To enable a tileset on a wms layer:

  1. Add any WMS to uDig
  2. Right click on the WMS resource
  3. Go to the Layer Properties
  4. Click Tileset Properties - on the left menu
  5. Check the checkbox - "Use a Tileset for this resource
  6. Change any of the default items you wish to use
