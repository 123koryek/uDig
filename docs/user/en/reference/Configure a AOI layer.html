Configure a AOI layer

You can use layer preferences to configure a polygon layer as a background "AOI" layer for use with the AOI Selection tool.

  1. Right click on the layer in the Layer view
  2. Click Properties in the context menu
  3. Click Interactions in the list on the left
  4. Click the Background radio button
  5. Click the AOI radio button
