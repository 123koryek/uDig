To attach a document to a layer:

  1. Open the Documents view
  2. Select the layer
  3. The list of documents associated with this layer will be shown in the Documents view
  4. Select the layer’s folder or a document inside the folder
  5. Press the Attach... button, a dialog will be provided to input details

To attach a document to a feature:

  1. Open the Documents view
  2. Select a feature using the Feature Selection tool
  3. The list of documents associated with this feature will be shown in the Documents view
  4. Select the feature’s folder or a document inside the folder
  5. Press the Attach... button, a dialog will be provided to input details
